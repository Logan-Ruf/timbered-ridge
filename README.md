# Launching Dev Environment

## HTTPS Proxy

Install `mkcert` and generate a certificate for `localhost`:

```bash
brew install mkcert
mkcert -install
mkcert localhost
```

For windows plebs use this [guide.](https://www.storyblok.com/faq/setup-dev-server-https-windows)
